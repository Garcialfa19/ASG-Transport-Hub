# **App Name**: ASG Transport Hub

## Core Features:

- Route Display: Display available bus routes with details such as name, duration, and fare.
- Alert Notifications: Show active alerts for passengers fetched from Firestore.
- Schedule Viewer: Allow users to view detailed bus schedules in a zoomable modal.
- Admin Authentication: Secure admin dashboard with Firebase Authentication for content management.
- Route Management: Enable admins to add, edit, and delete bus routes, including image uploads.
- Driver Management: Enable admins to manage drivers, assigning them to routes and updating their status and comments.
- Alert Management: Enable admins to create, edit, and delete alerts displayed to passengers.

## Style Guidelines:

- Primary color: Strong blue (#3B82F6) to evoke trust and reliability.
- Background color: Light gray (#F5F5F4) for a clean, modern look.
- Accent color: Light blue (#BEE3F8) for highlights and interactive elements.
- Body and headline font: 'PT Sans', a sans-serif with a modern and approachable feel.
- Use `lucide-react` icons for a consistent and modern look across the app.
- Employ a clean, grid-based layout with a focus on readability and ease of navigation.
- Subtle animations for transitions and loading states to enhance user experience.
